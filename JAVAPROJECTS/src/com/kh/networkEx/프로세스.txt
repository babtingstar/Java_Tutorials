Process

프로그램이 실행될 때 운영체제에서 메모리 공간을 할당받아 실행(void main())
각 프로세스는 독립된 메모리 공간을 가짐
다른 프로세스와 메모리 영역을 공유하지 않음

프로세스와 스레드의 관계
 - 프로세스는 스레드를 포함한다 : 하나의 프로세스는 적어도 하나의 스레드(작업)을 포함
 - 스레드(작업)는 그 프로세스(작업 실행하는 공간)에서 실행 흐름 담당

 하나의 프로세스의 여러 스레드 : 여러 스레드(여러 작업)가 하나의 프로세스(공간) 내에서
 동시에 각자의 작업을 실행하고 공유함
 동시에 작업을 수행할 수 있음

 프로세스 : 독립적으로 실행되는 프로그램의 인스턴스, 각기 다른 메모리 공간을 가짐
 스레드 : 프로세스 내에서 실행되는 작은 단위로, 프로세스 자원을 공유하며, 동시에 여러 작업 처리 가능
